Overview

This application allows users to manage tasks efficiently by organizing them into different categories such as All Tasks, In Progress, and Done. Users can create new tasks, add descriptions to tasks, mark tasks as done, delete tasks, and change the status of tasks from done to in progress.

Features

User Management: Create new users to manage tasks.

Task Management:

Add Task: Users can add new tasks along with descriptions.
![Todo Interface](./Photos/Screenshot%202024-03-24%20223832.png)
View Tasks: Tasks are categorized into All Tasks, In Progress, and Done for easy organization and management.
Mark Task as Done: Click on a task in the All Tasks list to mark it as done.
![Task Done](./Photos/Done%20Photo.png)
Delete Task: Click on the delete icon visible on the task to delete it.
![Task Delete](./Photos/Delete_Photo.png)
Change Task Status: To change a task from All Tasks to In Progress or vice versa, click on the timer symbol that appears on the task.
![Task Status Change](./Photos/Timer%20button.png)

Getting Started:

Clone the repository to your local machine.
Open the project in your preferred IDE.
Build and run the application.
Follow the on-screen instructions to create new users, add tasks, manage task statuses, and perform other actions.

Technologies Used:
C# programming language
WPF framework for Windows/Desktop application development
MVVM (Model-View-ViewModel) architecture
JSON file for data persistence
